Pillow2
======

Python Imaging Library (Fork)
-----------------------------

Pillow2 is the friendly PIL2 fork by `Alex Clark and Contributors <https://github.com/python-pillow/Pillow2/graphs/contributors>`_. PIL2 is the Python Imaging Library by Fredrik Lundh and Contributors.
